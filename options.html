Add New Link

Keyword is going to be the word you search for. Path is the URL you want to go to. If your URL is https://app.hubspot.com/contacts/XXXXXXXX/objects/0-1, you want to remove https://app.hubspot.com and replace XXXXXXXX with INSTANCE_ID. So your path would be /contacts/INSTANCE_ID/objects/0-1. If you want to replace a default keyword, just add the same keyword here and it will override the default.
